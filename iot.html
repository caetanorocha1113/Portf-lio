Automação Industrial & Urbana

CONTROLE DE SENSORES E ATUADORES

Este projeto consistiu no desenvolvimento de dois sistemas críticos de automação utilizando a plataforma Arduino. O primeiro módulo simula uma Estufa Inteligente, onde implementei uma lógica de controle térmico que realiza a leitura constante de um sensor de temperatura. O sistema foi programado para ativar um cooler de resfriamento aos 30°C e, em caso de superaquecimento crítico (acima de 50°C), disparar um alarme sonoro via Buzzer e alerta visual com LED vermelho.

O segundo módulo foca na Mobilidade Urbana, através da criação de um semáforo automatizado com temporização precisa via código (5s verde, 2s amarelo, 5s vermelho). A atividade permitiu o domínio da sintaxe em C para hardware, manipulação de portas PWM e compreensão de fluxogramas de automação.

Milesight IA Redes 4G Cloud Computing Smart City

Sistema Sente

MONITORAMENTO DE TRANSPORTE EM TEMPO REAL

Em colaboração com Larissa Amorim, projetamos o Sente, uma solução de Internet das Coisas voltada para resolver o caos da superlotação no transporte público de Florianópolis. O sistema utiliza sensores de visão computacional Milesight VS13x instalados nas portas dos veículos para realizar a contagem bidirecional (entradas e saídas) de passageiros com precisão absoluta.

A arquitetura do projeto prevê o envio desses dados via conexão 4G para uma infraestrutura na nuvem, onde o cálculo de ocupação percentual é realizado instantaneamente. O resultado é entregue ao usuário final através de um aplicativo móvel, permitindo que o passageiro saiba se há lugares vagos antes mesmo do ônibus chegar ao ponto, otimizando o fluxo urbano e a gestão das empresas de transporte.
